性能优化

发表于 2017-04-14 | 分类于 js

##移动端优化

加载优化

1. 合并css javascript
2. 合并小图片 css sprite
3. 缓存一切可以缓存的资源
4. 使用长cache
5. 使用内联样式和js
6. 压缩html css js
7. 使用Gzip
8. 使用首屏加载
9. 使用按需加载
10. 使用滚屏加载
11. 减少cookie
12. 使用loading
13. 异步加载第三方资源
14. 避免重定向

Css优化

1. css写在头部 js写在尾部或者异步加载
2. 避免图片和iframe空的src
3. 避免重设图片的大小
4. 避免在html中使用style属性
5. 避免css表达式
6. 移除空的css规则
7. 值为0的时候不带单位
8. 标准化浏览器前缀
9. 选择器不要过长

图片优化

1. 使用智图 压缩图片
2. 使用css3 svg iconfont dataUrl 代替图片
3. 使用scrset 图片响应式尺寸
4. webp优于jpg
5. png8 优于 gif
6. 首次加载不大于1014kb
7. 图片宽度不大于640

脚本优化

1. 减少重绘和重排
2. 缓存dom选择和计算
3. 缓存列表length
4. 使用时间代理批量处理事件绑定
5. 尽量使用id获取元素
6. 使用touchstart touchend 替代click

渲染优化

1. html使用viewport
2. 减少dom节点
3. 使用requestAnimateionFrame代替setTimeout
4. 尽量使用css3动画
5. 使用css3 transition 3d transform 来触发GPU渲染

浏览器缓存

1. 开启强缓存
